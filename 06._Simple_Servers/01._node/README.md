# How to setup Express

```bash
$ npm init -y
$ npm i express
```

Then add the following key-value pair in top-level of `package.json`:

```json
"type": "module"
```

Then you are ready to do the follwing in the `app.js` file:

1. Import
2. Initialize
3. Start the server by making it listen on a port
4. Create routes
